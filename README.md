# mybatis-dict-helper
A mybatis plugin to solve dictionary functionality

一个用来解决字典功能的mybatis插件

集成即可简单使用


### 初始版本功能列表


| 功能 | 是否支持 |
| --- | --- |
| 字典注解 | 支持 |
| 字典表DB查询 | 支持 |
| 字典记录缓存 | 支持 |
| 字典缓存超时清除 | 支持 |
| 无spring集成使用 | 支持 |
| SpringBoot集成使用 | 支持 |
